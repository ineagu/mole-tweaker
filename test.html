Instructions

  1. Install the Mole Tweaker extension in developer mode
  2. Open this test page
  3. Click the extension icon in your browser toolbar
  4. Toggle "Enable for this page" to ON
  5. Adjust the parameters (quality, format, etc.)
  6. Click "Apply Changes"
  7. Check the browser's developer tools (Network tab) to see the modified URLs being requested
  8. You can also inspect the img elements to see the modified src attributes

Expected Behavior:

  • Only Optimole URLs (containing .i.optimole.com) should be modified
  • Regular images should remain unchanged
  • Modified images should have a data-mole-tweaker-modified="true" attribute
  • When you disable the extension, original URLs should be restored
  • Settings should persist when you reload the page
